HTML Tags

"Tags" provide web browsers with instructions about the web page, such as where to display images, and how the document is structured.

Headings : <h1> - <h6>

This tag is used to define HTML headings in different sizes.

Example:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

Paragraph : <p>

The <p> tag defines a paragraph. Browsers automatically add a single blank line before and after each paragraph.

Example:

<p> This is some text in a paragraph. </p>

Line Break Tag : <br>

The <br> tag inserts a single line break.

Example:

<p>To force<br>line breaks<br>in a text,<br>use the br<br>element.</p>

Horizontal Line : <hr>

The <hr> tag defines a thematic break in an HTML page.

Example:

<p>HTML is the standard markup language for creating Web pages.</p>
<hr>
<p>CSS is a language that describes how HTML elements are to be displayed.</p>

Italics Tag

The <i> tag defines a part of text in an alternate voice or mood. The content is usually displayed in italics.

Example:

<p> <i>Lorem ipsum</i> is the most popular filler text in history.</p>

Bold Tag : <b>

The <b> tag is used to draw attention to text by making it bold, without implying importance.

Example:

<p>This is a <b>bold</b> statement.</p>

Image Tag : <img>

The <img> tag is used to embed images. It requires the 'src' and 'alt' attributes.

Example:

<img src="image.jpg" alt="Example Image" width="300">

Anchor Tag : <a>

The <a> tag defines a hyperlink to navigate between pages.

Example:

<a href="https://www.example.com">Visit Example</a>

List Tags : <ul>, <ol>, <li>

The <ul> tag creates an unordered list, <ol> creates an ordered list, and <li> defines list items.

Example:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

Div Tag : <div>

The <div> tag is a block-level container used to group elements for styling or scripting.

Example:

<div class="container">Content goes here</div>

CSS Properties

Color

Sets the color of the text.

p {
  color: blue;
}

Background-color

Sets the background color of an element.

div {
  background-color: white;
}

Font-family

Specifies the typeface for the text.

body {
  font-family: 'Overpass', sans-serif;
}

Controls the size of the text.
