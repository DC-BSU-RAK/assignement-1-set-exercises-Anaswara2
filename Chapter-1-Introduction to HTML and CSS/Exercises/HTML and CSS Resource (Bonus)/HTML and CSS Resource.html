HTML Tags

"Tags" provide web browsers with instructions about the web page, such as where to display images, and how the document is structured.

Headings : <h1> - <h6>

This Tag is used to define HTML headings in different sizes.

example:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

Paragraph : <p>

The <p> tag defines a paragraph.Browsers automatically add a single blank line before and after each </p> element.

example:

 <p> This is some text in a paragraph. </p>

Line Break Tag : <br>

The <br> tag inserts a single line break.

example:

 </p>To force <br> line breaks<br> in a text,<br> use the br<br> element.</p>

Horizontal Line : <hr>

The <hr> tag defines a thematic break in an HTML page

example:

 <p>HTML is the standard markup language for creating Web pages.</p> 
 <hr> 
 <p>CSS is a language that describes how HTML elements are to be displayed on screen, paper, or in other media.</p>;

Italics Tag

The <i> tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.

example:

 <p> <i> Lorem ipsum</i> is the most popular filler text in history.</p>

Bold Tag : <b>

The <b> tag is used to draw attention to text by making it bold, without implying importance.

example:

<p>This is a <b>bold</b> statement.</p>

Image Tag : <img>

The <img> tag is used to embed images in an HTML page. It requires the 'src' and 'alt' attributes.

example:

<img src="image.jpg" alt="Example Image" width="300">

Anchor Tag : <a>

The <a> tag defines a hyperlink, which is used to link from one page to another.

example:

<a href="https://www.example.com">Visit Example</a>

List Tags : <ul>, <ol>, <li>

The <ul> tag creates an unordered list, <ol> creates an ordered list, and <li> defines each list item.

example:

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

Div Tag : <div>

The <div> tag is a block-level container used to group elements and apply CSS styling or JavaScript functions.

example:

<div class="container">Content goes here</div>

CSS Properties

Color

The color property sets the color of the text.

p {
  color: blue;
}

Background-color

This property sets the background color of an element.

div {
  background-color: white;
}

Font-family

Specifies the typeface (font) of the text.

body {
  font-family: 'Overpass', sans-serif;
}

Font-size

Controls the size of the text.

h1 {
  font-size: 30px;
}

Margin

Defines space around elements outside the border.

div {
  margin: 20px;
}

Padding

Defines space inside the border, between the border and the content.

div {
  padding: 15px;
}

Border

Used to set the border around an element.

img {
  border: 2px solid black;
}

Text-align

Specifies the horizontal alignment of text.

p {
  text-align: center;
}
